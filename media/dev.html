Cette version personnalisé de monsieur Patate faite en Javascript s'inscrit dans le cadre d'un projet de classe en duo. Maps Sud

Dans le cadre de mon stage à l'Espace Sud, j'ai travaillée sur la recherche et la création d'extensions pour le site web afin de le rendre dynamique. Ici le but était de visualiser les communes de l'agglomération, j'ai créee cette carte interactive en Python qui, lorsque l'on selectionne une commune elle nous emmène un article qui détaille les informations sur celle-ci tel un mini guide touristique.

Maps Sports

Pour ce même stage, j'ai cette fois-ci proposé une solution pour que les utilisateurs puissent visualiser tous les sports de l'agglomération de manière plus fluide. Pour ce faire j'ai pu créer des cartes intéractives en Python triant les sports par catégories , chaque sport est visible à l'aide d'un pins sur la carte contenant ses informations (nom, numéro de téléphone, email) .

Frise Carte

Pour rendre le site du Tour des Yoles plus dynamique et faciliter la visibilité j'ai pu ajouter une frise faite en Javascript regroupant les cartes des Etapes. chaque jour la carte du parcours de l'étape était ajouté dans un item orange, jusqu'a maintenant ces cartes sont disponibles , cette extension permet à la fois de porter l'attractivité du site mais aussi de stocker les cartes .
